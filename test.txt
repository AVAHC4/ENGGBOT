ENGGBOT - AI Study Assistant Platform Test File

This test file is part of the ENGGBOT project, an advanced AI study assistant platform that leverages course materials to provide academically accurate responses to student queries.

Key Features:
- AI-powered chat interface with DeepSeek models
- Google OAuth authentication
- Modern Next.js 14 frontend with TypeScript
- Animated landing page with Framer Motion
- Voice input support via microphone integration
- Thinking mode for detailed AI reasoning process

Project Status: Version 1.0.1
Architecture: Next.js + TypeScript + Supabase + Chutes AI

This file can be used for testing various components of the platform including:
- File processing capabilities
- AI response generation
- Database operations
- Authentication flows

For development setup, run:
npm install
npm run dev

Environment variables required:
- CHUTES_API_KEY
- GOOGLE_CLIENT_ID
- GOOGLE_CLIENT_SECRET
- NEXT_PUBLIC_SUPABASE_URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY
