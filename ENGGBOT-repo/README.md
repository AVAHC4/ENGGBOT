# ENGGBOT - Engineering Assistant AI

ENGGBOT is an AI-powered engineering assistant designed to help with coding, technical problem-solving, and engineering challenges.

## Features

- **Text Generation Effect**: Smooth text generation with proper link handling
- **Google Authentication**: Secure login with Google
- **Responsive UI**: Works on desktop and mobile devices
- **Code Block Formatting**: Proper syntax highlighting for code snippets
- **Real-time AI Responses**: Fast responses to engineering queries

## Getting Started

1. Clone this repository
2. Navigate to the src directory
3. Install dependencies: `npm install`
4. Set up environment variables (see .env.example)
5. Run the development server: `npm run dev`

## Technical Stack

- Next.js
- React
- Tailwind CSS
- Framer Motion for animations
- DeepSeek AI integration
